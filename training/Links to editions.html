Links to Specific Pages

If you need to send a link to a specific page in the ePDO to someone, add &admin=true to end of the URL to enter admin mode. Alternatively, you can use the admin links above. At the bottom of each page when in Admin Mode, you will find links to that specific page for each edition.

Avoid sending page-specific links to students who have not yet completed the training, since they may not realize that they need to go back and read the previous pages.

Admin mode gives you links for each edition you started in, so it doesn't really matter which edition you start with, except that you will be certain to see the content you want to see.

Print Mode

You can print an individual page by simply navigating to that page and printing as you would any web page. If you'd like to print the entire Survival Guide, you'll need to navigate to the special Print All page, by adding &nav=Print to the URL--make sure that there is no other navigation (i.e. nav=) bit in the URL when you do this. Or, just use the Print All link for the edition you're interested in above.

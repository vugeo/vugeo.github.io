INLINE SVG MAPS
---------------------

The responsive .svg maps in this file are not included in Twine with their src path. 
Instead, their source codes are individually added into the Twine Story as separate passages. 

To display the maps on the desired passage, use the <<include>> macro.
	e.g. 	:: Blank World Map
		<svg...>...</svg>

		:: Regional PreDeparture Orientation 
		Please select your region on the map below:
		<<include "Blank World Map">>
